# Up & Away - Drone Website

A multi-page drone e-commerce website built as part of my LV4 Multimedia course.

## Pages
- **Home** – Welcome page with banner and intro
- **Products** – 6 drones available to browse and buy
- **Gallery** – Image slideshow using Fotorama
- **Contact** – Contact form with validation

## Built With
- HTML5
- CSS3
- JavaScript / jQuery
